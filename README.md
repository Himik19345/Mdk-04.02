# BukHottel - Система онлайн-бронирования гостиничных номеров

Веб-платформа для поиска, выбора и безопасного бронирования отелей.

## О проекте
**BukHottel** — это сервис, который позволяет туристам быстро находить комфортное размещение с прозрачным ценообразованием, а менеджерам отелей — управлять номерным фондом и бронированиями.

## Основные возможности
* Каталог отелей с умными фильтрами (цена, звездность, рейтинг)
* Сквозное оформление бронирования и выбор дополнительных услуг
* Личный кабинет с историей заказов и статусами
* Интеграция службы поддержки (онлайн-чат с ИИ-ассистентом)
* Административная панель для управления пользователями и ролями

## Технологии
* **Frontend:** HTML5, CSS3, JavaScript
* **Архитектура:** 4 уровня доступа (Гость → Менеджер отеля → Разработчик → Владелец)
* **Модель разработки (МЖЦ):** Scrum (Agile)

## Быстрый запуск

**Вариант 1: Через установщик (рекомендуется)**
1. Скачайте файл `mdk.msi` из корня репозитория.
2. Запустите файл и следуйте инструкциям установщика. 
3. На рабочем столе (или в выбранной папке) появятся файлы проекта. Запустите главную страницу.

**Вариант 2: Запуск без установки**
1. Скачайте исходный код проекта.
2. Откройте папку `МДК` и запустите файл `index.html` в любом современном браузере.
